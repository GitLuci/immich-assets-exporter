# Immich Assets Exporter

## Project Objective
### Explorate a simple way to create redundant HDs and security backups from your Immich galery


## Project set up
### Edit the .env-example and add your [Immich API Key](https://api.immich.app/getting-started) and add your Server URL
### After that you will be able to run the code.
