# Immich Assets Exporter

## Project Objective
### Explorate a simple way to create redundant HDs and security backups from your Immich galery


## Project set up
### Edit the .env-example and add your [Immich API Key](https://api.immich.app/getting-started) and add your Server URL
### After that you will be able to run the code.
### You can run the tests file to check if you are able to access everything, is a simple way to test your permissions, can have some bug, anything let me know in the comments.
## How to run
### After clone the project and set the env variables you can run the streamlit Frontend using the command from the root of project
```bash
python -m streamlit run screens/home.py
```
